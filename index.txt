Connecting to EC2 Using AWS CloudShell
Create the EC2 instance using a key pair (.pem).
Log in to AWS Console and click on the CloudShell icon (terminal icon on the top-right).
Once CloudShell opens, go to AWS Services → EC2.
Select your EC2 instance and click Connect.
Choose the SSH client tab.
Copy the ssh -i command shown there.
Paste the command in CloudShell and try to run it. ❌ It will fail with an error like: “key pair is not available / no such file or directory” (because the key file is not yet in CloudShell).
Now, in CloudShell, click on the “+” (Actions) icon on the right side.
Select Upload file and upload the key pair (.pem) from your local machine.
After upload, copy the ssh -i command again from EC2 → Connect → SSH client.
Paste and execute the command in CloudShell.
✅ You are now successfully connected to the EC2 instance.

Step-by-Step: Create Windows EC2 Instance and Connect via RDP
Step 1: Launch a Windows EC2 Instance
Log in to your AWS Management Console.
Navigate to EC2 → Instances → Launch Instances.
Select an Amazon Machine Image (AMI):
Choose a Windows Server AMI (e.g., Windows Server 2019 or 2022).
Choose an Instance Type (e.g., t2.micro for free tier or as per your requirement).
Configure Instance Details as needed.
Add Storage if required (default is usually sufficient).
Configure Security Group:
Ensure RDP port 3389 is allowed.
Source can be My IP (for security) or a wider range if needed.
Select or create a new Key Pair:
Download the .pem file (keep it safe; it’s needed to decrypt the password).
Click Launch Instance.

Step 2: Get the Public DNS / IP
After the instance is running, go to EC2 → Instances.
Select your instance and note the Public DNS (IPv4) or Public IP.

Step 3: Retrieve the Windows Administrator Password
Select your instance → Click Connect → RDP Client.
Click Get Password.
Upload your Key Pair (.pem) file.
Click Decrypt Password.
Copy the Administrator password shown.

Step 4: Connect via RDP
On your local Windows machine, open Remote Desktop Connection (mstsc).
In the Computer field, enter the Public DNS or IP address of the EC2 instance.
Click Connect.
In the login prompt:
Username: Administrator
Password: Paste the decrypted password from Step 3.
Click OK.

Step 5: You’re connected
Your Windows EC2 instance desktop should appear.
You can now use it as a normal Windows machine.
